Бот для получения расписания с сайта РГГУ по данным пользователя.

Содержит файлы:
1. bot_body_rsuh.py - файл с телом бота (он же главный файл (main))
 - написан на библиотеке telebot
  
2. user_database.json - json, который используется в качестве базы данных для хранения информации о пользователях
  - username - КЛЮЧИ бд, имя пользователя в тг (строка)
  - form - форма обучения (строка)
  - course - курс (строка: Курс + номер)
  - speciality - специальность (строка)
  
3. get_speciality.py - файл для парсинга всех специальностей с сайта (нужно запускать раз в учебный год для обновления списка специальностей)
  - используются selenuim, bs4, requests, json
  
4. RSUH_parsing_raspis.py - файл для парсинга расписания с сайта по данным пользователя
- используются bs4, requests (Саша, спасибо<3)

5. specialities.json - json со всеми специальностями, представленными на сайте РГГУ
 - ключ: Форма обучения + Курс
 - значение: список всех специальностей по ключу
